## コア成長システムの理解

このゲームの成長要素は、「レベル」、「ランク」、「クラス開発」の3つの柱に分かれています。レベルだけが強さの指標である他の多くのRPGとは異なり、このゲームの「ランク」（EからS+まで）は、アクセスできるコンテンツの難易度や、受け取れる報酬の質を決定します。

### ステータス配分とマイルストーン

レベルアップするたびに、ステータスポイントを1ポイント獲得できます。これらのポイントは、ビルドの自由度を決める生命線です。ポイントは4つの主要属性に振り分けることができます。特定のステータスが100ポイントに達すると、プレイスタイルを決定づけるユニークなパッシブパークが解放されます。

| ステータス | 主なメリット | 100ポイント時のパーク |
| :--- | :--- | :--- |
| **素早さ (Agility)** | 移動速度とダッシュ距離が上昇。 | **テレポート:** 高速のブリンク能力を解放。 |
| **筋力 (Strength)** | 物理ダメージと通常攻撃（M1）の威力が上昇。 | **体勢崩し:** 相手のポスチャー（体勢）に与えるダメージが大幅に増加。 |
| **体力 (Vitality)** | 最大体力（HP）が上昇。 | **レジリエンス:** パッシブな体力回復ブースト。 |
| **知力 (Intelligence)** | マナ（MP）とスキルダメージが上昇。 | **効率:** スキルのクールダウンとマナ消費を軽減。 |

> **💡 ヒント:** 序盤にポイントを分散させすぎるのは避けましょう。まずは一つのステータスを100にすることに集中し、ゲーム性を変えるパークを解放してから他のステータスに手を広げるのが得策です。

## クラスとパスの選択

このゲームの最も革新的な機能の一つは、「**クラス**」と「**パス**」が分離している点です。クラスが戦闘スキルを決定する一方で、パスは冒険の中でどのようなコンテンツに焦点を当てるかを決定します。

### 6つの戦闘クラス
早期アクセス版では合計6つのクラスが用意されています。各クラスには、クラスポイントを投資することで解放される7つのユニークなスキルがあります。

1.  **戦士 (Warrior):** 高い物理バースト能力を持つ、バランスの取れた前衛ファイター。
2.  **タンク (Tank):** 敵の足止めと、膨大なダメージの吸収に特化。
3.  **暗殺者 (Assassin):** 高い機動力とクリティカル性能を持ち、1対1に優れる。
4.  **魔導士 (Mage):** 広範囲攻撃（AoE）を得意とする遠距離属性攻撃。
5.  **ヒーラー (Healer):** HPやマナの回復を提供し、グループプレイには欠かせない存在。
6.  **ストライカー (Striker):** 素早いコンボ攻撃に特化した、テンポの速い格闘家。

### 3つのゲームプレイパス
冒険の開始時に、パスを選択する必要があります。これはクラスとは別のシステムで、この世界におけるあなたの「職業」を定義します。

| パス | 焦点 | 主な活動 |
| :--- | :--- | :--- |
| **会長 (Chairman)** | PvE & 経済 | 救助任務の指揮、希少鉱石の採掘、リソース管理。 |
| **暗殺者 (Assassin)** | PvP & 賞金稼ぎ | サーバーの契約掲示板を通じて他プレイヤーを狩り、報酬を得る。 |
| **ハンター (Hunter)** | ボス & 討伐 | 高ランクのボスミッションへの挑戦や、エリートゲートの攻略。 |

## クエストとゲートの仕組み

強くなるためには、クエストボードを利用する必要があります。ミッションはEからSまでランク付けされており、多様な目的設定によりゲームプレイがマンネリ化することはありません。

### ミッションの種類
*   **調査 (Investigate):** 非戦闘重視のミッション。地上に隠された手がかりやアイテムを探します。
*   **ゲートクリア (Clear Gate):** クラシックなダンジョン体験。ポータルに入り、すべてのモブを倒し、ボスを撃破してゲートを閉じます。
*   **討伐 (Subjugation):** 特定のアリーナにテレポートし、1体または2体の強力な敵と戦います。

### ゲートシステム
ゲートは、上位ティアの戦利品や設計図を入手するための主な手段です。特別な**レッドゲート**や**シークレットダンジョン**を含む、8つの異なるゲートランクが存在します。シークレットダンジョンにアクセスするには、デイリークエストの完了やレッドゲートからのレアドロップで入手できる「**シークレットキー**」が必要です。

> **⚠️ 警告:** ソロでゲートに入る場合は、難易度の急上昇に注意してください。ボスはパリィメカニクスや、ライフを一気に削り取る高威力の範囲攻撃を持っています。

## デイリートレーニング：偉大さへの道

「俺だけレベルアップな件」から着想を得たこのゲームには、単なるボタン連打よりもインタラクティブなデイリークエストシステムがあります。ペナルティを回避し、大量の経験値ブーストやシークレットキーを獲得するために、毎日4つの身体的タスクを完了する必要があります。

*   **腕立て伏せ:** リズムに合わせたミニゲーム。
*   **腹筋:** 動くバーの中でタイミングよくクリックする必要があります。
*   **スクワット:** 腹筋と同様、反応速度が試されます。
*   **ランニング:** ゲーム内の世界で特定の距離（例：400メートル）を走破する必要があります。

これらのタスクを完了することは、序盤で一気に5レベル上げるための最速の方法です。公式アップデートの詳細については、コミュニティイベントが開催される[Robloxグループ](https://www.roblox.com/groups/1)をチェックしてください。

## 装備、製作、そして特性

この**Reawakened早期アクセスガイド**を読み進めるうちに、素のステータスだけでは不十分であることに気づくでしょう。クラスを補完する装備が必要です。防具や武器は多くの場合「**設計図 (Blueprints)**」としてドロップされ、ゲート内で収集した素材（アンテナ、クモの糸、マナクリスタルなど）を使用して製作する必要があります。

### 特性 (Traits)：隠された力
特性は、二次的な能力やパッシブバフとして機能します。現在、ゲーム内には7つの特性があります。最も人気があるのは「**ゴリアテ (Goliath)**」特性で、身体の大きさとダメージ耐性を大幅に向上させます。特性はランダムに付与されるため、自分のビルドに合うものを見つけるためにリロールしたり、特定のコンテンツを周回したりする必要があるかもしれません。

| 特徴 | 説明 |
| :--- | :--- |
| **設計図** | ダンジョンの宝箱から発見。製作には特定の素材が必要。 |
| **採掘** | ゲート内でつるはしを使用し、装備アップグレード用の鉱石を収集。 |
| **ランク評価** | ランクNPCを訪れ、実力をテストして公式にランクを上げる。 |

## 早期アクセスのための戦闘のコツ

1.  **ダッシュを極める:** 'Q'キーでダッシュします。ほとんどのボスの攻撃には、赤い予兆範囲や特定の予備動作といった「テレグラフ」があります。直前でダッシュすることで命拾いすることが多いです。
2.  **体勢管理:** プレイヤーと敵の両方にポスチャー（体勢）バーがあります。短時間で十分なダメージを与えると敵の体勢を「崩す」ことができ、数秒間スタン状態にできます。
3.  **つるはしを活用する:** 「会長」パスでなくても、つるはしは持ち歩きましょう。ゲート内のマナクリスタル採掘は、通貨や製作素材を稼ぐための最も確実な方法の一つです。
4.  **回復は限定的:** ポーションを連打できる他のゲームとは異なり、ゲート内での回復は困難です。序盤に数ポイントを体力（Vitality）に振り、余裕を持たせておきましょう。

## よくある質問 (FAQ)

**Q: Reawakened早期アクセス版で武器を入手するにはどうすればいいですか？**

A: 通常は素手から始まります。最初のクラス専用武器（暗殺者なら短剣など）を入手するには、選択したクラスツリーに少なくとも5ポイント投資するか、Eランクゲートで見つかる設計図から基本的な武器を製作する必要があります。

**Q: デイリークエストを失敗するとどうなりますか？**

A: デイリートレーニングを完了できないと、「ペナルティ」が発生する可能性があります。多くの「俺レベ」系ゲームと同様に、これは高難易度の生存ゾーンに転送され、脱出するために押し寄せる敵の波を生き抜かなければならないことを意味します。

**Q: 後でクラスやパスを変更することはできますか？**

A: 現在、パスは序盤に行う重要な選択です。ショップのアイテムやレアドロップで「リセット」アイテムが入手できる可能性はありますが、最初から自分の好みのプレイスタイル（PvEかPvPか）に合ったパスを選ぶのが最善です。

**Q: EランクからSランクに上げるにはどうすればいいですか？**

A: ランク評価NPCを訪れる必要があります。NPCが現在のステータスとレベルを評価し、ランクアップの準備ができているか判断します。高ランクの装備には装備条件として最低ランクが設定されていることが多いため、ランクアップは不可欠です。
