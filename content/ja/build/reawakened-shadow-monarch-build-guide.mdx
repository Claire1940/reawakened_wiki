## 正しい開始パスの選択

Reawakenedでは、あなたの旅はパスの選択から始まります：暗殺者（Assassin）、ハンター（Hunter）、会長（Chairman）、または鑑定士（Evaluator）。影の君主の称号を目指す者にとって、**暗殺者**パスは一般的に最も効果的な出発点です。このパスは水篠旬（Sung Jin-Woo）の初期の成長を反映しており、高い機動力と素早い攻撃に焦点を当てています。

暗殺者パスをアンロックするには、暗殺者トレーナーと話し、PvP戦闘で他のハンターを排除して自分の価値を証明する必要があります。この攻撃的なスタートは、高ランクゲートのハイリスクな環境に備えるためのものです。

| パス | 主な焦点 | 最適な用途 |
| :--- | :--- | :--- |
| **暗殺者** | **敏捷性と速度** | 高DPS、PvP、ボス狩り |
| **ハンター** | **筋力と体力** | PvEタンク、ダンジョン攻略 |
| **ネクロマンサー** | **知能とマナ** | 影の召喚、クラウドコントロール |
| **会長** | **バランスの取れた統計** | サポート、ギルド運営 |

> **警告：** パスの選択は長期的なコミットメントです。ショップにクラスチェンジアイテムは存在しますが非常に高価なため、影の君主ビルドに集中したい場合は、早い段階で暗殺者またはネクロマンサーのパスを選んでください。

## 必須のステータス割り振り

あらゆる**Reawakened 影の君主ビルドガイド**の核心は、ステータス配分にあります。一般的なビルドとは異なり、影の君主はハイブリッドなアプローチを必要とします。影の軍団を維持するための十分な**知能**と、「起き上がれ（Arise）」コマンドを使用するまで生き残るための十分な**敏捷性**が必要です。

2026年のメタでは、最も成功しているプレイヤーは以下の順序でステータスを優先しています：

1.  **敏捷性（Agility）：** 移動速度と回避率を上げます。高ランクの召喚をアンロックする前に必要な暗殺者スタイルの戦闘において不可欠です。
2.  **知能（Intelligence）：** マナプールを拡大します。すべての影の召喚は時間の経過とともにマナを消費するため、これは影の君主の生命線です。
3.  **筋力（Strength）：** 短剣や小太刀による生の攻撃ダメージをブーストします。
4.  **体力（Vitality）：** HPとダメージ抵抗を増加させます。

| ステータス名 | 効果 | 君主の優先度 |
| :--- | :--- | :--- |
| **敏捷性** | **速度/回避** | **高** |
| **知能** | **マナ容量** | **極めて高** |
| **筋力** | **攻撃力** | **中** |
| **体力** | **HP/防御** | **中** |
| **感覚** | **クリティカル/認識** | **低** |

## デイリートレーニングと罰の領域

Reawakenedの最もユニークな側面の一つは、デイリークエストシステムです。強くなるためには、毎日決められた一連の身体トレーニングを完了しなければなりません。これらのクエストを怠ると進行が停滞するだけでなく、**罰の領域**に飛ばされる可能性があります。そこは、一定時間、無限に押し寄せるモブの中で生き残らなければならない危険なエリアです。

デイリートレーニングを完了するには、指定されたキー（通常は「T」）を押してエクササイズインターフェースを開始します。各セットを完了するには、入力を正確にタイミングよく行う必要があります。

*   **腕立て伏せ（25回）：** 基礎筋力を増加させます。
*   **腹筋（25回）：** 基礎体力を増加させます。
*   **スクワット（25回）：** 基礎敏捷性を増加させます。
*   **ランニング（10km）：** 全体的なスタミナと速度を増加させます。

これらのデイリータスクを完了すると、多大な経験値とステータスブーストが得られ、ダンジョン探索以外でレベルアップするための最も安定した方法となります。

## 影の召喚とスキルの相乗効果

**Reawakened 影の君主ビルドガイド**の真の力は「起き上がれ（Arise）」スキルにあります。この能力により、倒した敵の影を抽出し、永続的な軍団に加えることができます。ただし、すべての敵を影にできるわけではありません。ボスやエリートモブは抽出成功率が低く、これはあなたの**知能**と**感覚**ステータスに影響されます。

### 狙うべきトップティアの影

*   **イグリット（血赤の騎士）：** 高い防御力と強力な剣技を持つバランスの取れた影。ボスのヘイトを買うタンクとして優秀です。
*   **ベル（蟻の王）：** DPS影の頂点。ベルは驚異的なスピードを提供し、敵のアーマーを切り裂くことができます。
*   **タンク（氷の熊）：** その名の通り、ダメージ吸収専用の影です。高ダメージの魔法攻撃を行う敵に直面した際に使用しましょう。
*   **カイセル（ワイバーン）：** 空中機動力と大規模な範囲（AoE）火炎攻撃を提供します。

> **💡 ヒント：** クイックスロットには常に「マナポーション」をいくつか入れておきましょう。マナがゼロになると、アクティブなすべての影が即座に消滅し、無防備な状態になってしまいます。

## ゲート攻略戦略

ダンジョン、つまり「ゲート」はランク（EからS）によって分類されます。影の君主ビルドの場合、ソロでゲートをクリアすることが、影の抽出に必要なボスのトドメを確実に刺すための最善の方法です。

1.  **雑魚敵の掃討：** 影を使って小さなモブのヘイトを引き付け、その間にアーチャーやメイジなどの優先度の高いターゲットを仕留めます。
2.  **マナ管理：** 基本的なモブに対して軍団全員を召喚しないでください。ボス戦のために少なくとも50%のマナを温存しておきましょう。
3.  **抽出プロセス：** ボスを倒した後、「起き上がれ」を使用できる時間は限られています。3回失敗すると、影は永遠に消滅してしまいます。

## FAQ

**Q: Reawakened 影の君主ビルドガイドにおいて最も重要なステータスは何ですか？**

A: 生存のためには敏捷性が重要ですが、ゲーム終盤の影の君主ビルドにおいて最も決定的なステータスは**知能**です。これは、維持できる影の数と、戦闘中に影がアクティブでいられる時間に直接影響します。

**Q: 影の抽出に失敗することはありますか？**

A: はい。抽出の成功はモブのランクとあなた自身のステータスに基づいています。知能と感覚のステータスが高いほど、成功率は上がります。一部のレジェンダリー級の影は、基本成功率が10〜15%しかない場合もあります。

**Q: デイリークエストを逃すとどうなりますか？**

A: デイリートレーニング（腕立て伏せ、腹筋、スクワット、ランニング）を完了できなかった場合、罰の領域に転送されます。元の世界に戻るには、砂漠のムカデなどの危険が潜む場所で（ゲーム内時間で）4時間生き残らなければなりません。

**Q: このビルドではネクロマンサーパスよりも暗殺者パスの方が良いですか？**

A: 暗殺者パスはソロ戦闘能力と機動力に優れており、序盤の攻略に役立ちます。ネクロマンサーパスは召喚獣へのバフが強力です。多くのプレイヤーは暗殺者としてスタートし、レベル40に達してから影の君主の役割に移行することを好みます。
