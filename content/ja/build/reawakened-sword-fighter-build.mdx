## 剣術に最適なクラスの選択

Reawakenedには6つの異なるクラスがありますが、すべてのクラスがソードファイターに特化しているわけではありません。剣の効率を最大化するには、物理武器の補正と近接戦闘を補完するクラスを選ぶ必要があります。

**ウォーリアー（Warrior）**クラスはこのビルドの黄金律です。体力、攻撃力、速度のバランスが取れており、様々な戦闘シナリオにおいて最も汎用性の高い選択肢となります。しかし、ハイリスク・ハイリターンなプレイスタイルを好むのであれば、耐久力は犠牲になりますが、驚異的なスピードとコンボ性能を誇る**アサシン（Assassin）**クラスも選択肢に入ります。

| クラス | 戦闘スタイル | 主なメリット | スキル相性 |
| :--- | :--- | :--- | :--- |
| **ウォーリアー** | オールラウンダー | 体力と攻撃のバランス | PvEおよびPvPでの高い汎用性 |
| **アサシン** | スピード/バースト | 圧倒的な機動力 | 長時間の高火力コンボ |
| **タンク** | フロントライン | 圧倒的な耐久力 | 高い体勢値と防御力 |
| **ストライカー** | コンボ重視 | 素早い打撃 | バーストダメージのための連携攻撃 |

> **💡 ヒント:** 初心者はまずウォーリアークラスで敵のパターンを学び、慣れてきたらより高度な操作が求められるアサシンやストライカーに移行することをお勧めします。

## ソードファイターの最適なステータス配分

Reawakenedでは、レベルアップごとに1ポイントのステータスポイントが付与され、俊敏、筋力、体力、知力に投資できます。**Reawakenedソードファイタービルド**では、筋力と俊敏への均等な振り分けを基本とし、ボスのギミックを耐えられるだけの最小限の体力を確保するのが理想です。

このゲームには、特定のステータスが100ポイントに達するとゲームチェンジャーとなる能力が解放される独自の「パーク（Perk）」システムがあります。剣使いにとって、これらのマイルストーンへの到達は交渉の余地のない必須事項です。

### ソードファイターの核となるステータス

*   **筋力（Strength）:** 生のダメージ量、そしてより重要な「体勢ダメージ（posture damage）」を増加させます。筋力が100に達すると、敵のガードを崩す能力を大幅に強化するパークが解放されます。
*   **俊敏（Agility）:** 移動速度と攻撃速度を向上させます。俊敏が100に達すると、距離を詰めたりレッドゲートのボスから逃れたりするのに不可欠な**テレポート（Teleport）**能力が解放されます。
*   **体力（Vitality）:** 最大HPを増加させます。直接的なダメージステータスではありませんが、体力を軽視するとSランクダンジョンで一撃死する原因となります。

| ステータスマイルストーン | 解放されるパーク | ソードビルドへの影響 |
| :--- | :--- | :--- |
| **筋力 100** | 体勢マスタリー | 敵のガードを25%早く崩す |
| **俊敏 100** | テレポート | コンボ中の瞬時の位置取りが可能 |
| **体力 100** | 強靭な肉体 | 物理攻撃に対する被ダメージを軽減 |

## パス（道）システム：フォーカスの選択

旅の始まりに、3つの「パス」のうち1つを選択しなければなりません。この決定によって、最も頻繁に関わるコンテンツの種類が決まります。**Reawakenedソードファイタービルド**の場合、プレイヤーを狩るのが好きか、モンスターを狩るのが好きかによって選択が変わります。

1.  **ハンターパス（Hunter Path）:** ほとんどのソードファイターに推奨されるパスです。討伐任務やボス戦に焦点を当てています。剣ビルドは単体へのダメージとガード崩しに優れているため、ボス狩りは努力に対して最高の報酬をもたらします。
2.  **アサシンパス（Assassin Path）:** 対人戦を楽しみたい場合、このパスではプレイヤーへの暗殺依頼を受けることができます。ソードファイターの高い機動力は、オープンワールドでのメイジやヒーラークラスにとって悪夢となるでしょう。
3.  **チェアマンパス（Chairman Path）:** PvEと資源収集に焦点を当てています。救助任務を率いたり、高ティアの装備をクラフトするために希少な鉱石を採掘したりすることを好む場合に選んでください。

## 装備と特性：刃を研ぎ澄ます

装備はビルドの最終的なレイヤーです。どんな装備でも機能する一部のRPGとは異なり、Reawakenedには選択したクラスに特化した装備が存在します。ソードファイターとしては、物理ダメージとスタミナ回復にボーナスを与える防具セットを探すべきです。

### 特性（Traits）：秘められた力
特性は、戦闘能力を劇的に変化させる二次的なパワーとして機能します。物理ビルドで最も切望される特性の一つが**ゴリアテ（Goliath）**です。

*   **ゴリアテ特性:** 物理攻撃力と生存率を大幅に向上させます。これにより、ソードファイターは怯むことなくボスと殴り合うことが可能になります。
*   **スピードスター特性:** アサシンベースの剣ビルドに理想的で、回避フレームと移動速度に特化しています。

> **警告:** デイリークエストを無視しないでください！腕立て伏せやスクワットのミニゲームは、装備のステータスを補完する不可欠な成長要素を提供します。

## 進行戦略：EランクからSランクへ

**Reawakenedソードファイタービルド**を成功させるには、構造化された進行ルートを辿る必要があります。ゲームには8つの異なるゲートランクがあり、適切な準備なしに高ランクダンジョンに突入すれば失敗は免れません。

### レベリングのマイルストーン

1.  **レベル1-20 (E～Dランク):** クエストボードに専念してください。「調査」や「ゲートクリア」のミッションを完了し、筋力と俊敏の基礎を築きます。
2.  **レベル21-50 (C～Bランク):** 100ポイントのステータスマイルストーンに集中し始めます。シークレットダンジョンのボスを確実に倒せるよう、まずは筋力を優先しましょう。
3.  **レベル51+ (A～S+ランク):** この段階でビルドを完成させます。レッドゲートを周回し、ウォーリアーやアサシンのスキルを補完するレジェンダリー級の武器や防具を集めます。

| ゲートランク | 難易度 | 推奨ビルド段階 |
| :--- | :--- | :--- |
| **E - D ランク** | 低 | 基本クラススキルと初期装備 |
| **C - B ランク** | 中 | 1つのステータスマイルストーン（100点）到達 |
| **A - S ランク** | 高 | 2つのステータスマイルストーンとゴリアテ特性 |
| **レッドゲート** | 極限 | 最大強化装備、最適化された特性、フルスキルツリー |

## FAQ

**Q: Reawakenedソードファイタービルドで最初に最大化すべきステータスは何ですか？**

A: 一般的にはまず**筋力（Strength）**を100にすることをお勧めします。体勢ダメージのボーナスは、ゲーム序盤から中盤のゲートに登場するボスのガードを崩すために不可欠であり、中断されることなく大ダメージを与えることが可能になります。

**Q: ウォーリアーのプレイスタイルが合わない場合、クラスを変更できますか？**

A: はい、特定のアイテムや通貨が必要になりますが、クラス変更は可能です。ウォーリアーが遅すぎると感じる場合は、剣に特化したステータスを維持したまま、**ストライカー**や**アサシン**に切り替えてビルドを転換することができます。

**Q: 進行においてデイリークエストは必須ですか？**

A: 厳密に必須ではありませんが、強く推奨されます。腕立て伏せ、腹筋、スクワットのミニゲームは一貫したステータス成長をもたらし、何時間もゲートを周回することなくランク間の実力差を埋める最も信頼できる方法の一つです。

**Q: ソロプレイヤーに最適なパスはどれですか？**

A: ソロプレイヤーには**ハンターパス（Hunter Path）**が理想的です。ボスや少数の敵グループと戦う討伐ミッションに焦点を当てており、ソードファイターの高い単体ダメージ性能と完璧に合致しています。
